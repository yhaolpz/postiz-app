# 让 Agent 自动炒股，哪些动作必须交还给人？

成片旁白以 [`narration.txt`](./narration.txt) 为唯一文本源，中文语音固定使用 `zh-CN-YunxiaNeural +20%`。

## 内容结构

1. 前言：真实损失与三级权限矩阵
2. 机制：按动作后果划分自动、确认、阻断
3. 诊断：分清事实、观点、未核实消息与外部指令
4. 行动：把权限矩阵、硬限制、日志和失败路径写进工作流
5. 总结：把决定在高风险动作前交还给人

贯穿场景是一个经营一人公司的人使用 Agent 管理投资研究和模拟交易。内容只用于讲解 Agent 权限，不构成投资建议，也不承诺收益。
