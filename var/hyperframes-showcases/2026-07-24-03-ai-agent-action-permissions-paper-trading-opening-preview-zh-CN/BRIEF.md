# Tiny Agent 中文开头两分钟预览

## 目标

制作现有“AI 智能体行动权限”主题的中文开头预览，供完整重制前检查。预览是未来完整版的前两分钟，不是独立短视频，不覆盖既有中文成片，也不进入任何发布流程。

## 观众与主旨

- 观众：正在把真实电脑工作交给 AI Agent 的普通用户、一人公司老板和创作者。
- 主旨：Agent 可以自动研究、计算、写草案和执行模拟操作；一旦要动真实资金、传敏感数据或执行不可逆动作，必须进入明确确认或阻断。
- 固定母案例：一人公司老板把模拟投资账户交给 Agent 管理。
- 可复用工具：自动 / 确认 / 阻断三级行动权限矩阵。

## 内容边界

- 只讲模拟投资，不提供个性化证券建议，不展示真实账户、真实凭据或真实下单。
- 不承诺自动赚钱、稳定收益或跑赢市场。
- 来源事实继续来自 OpenAI《Designing AI agents to resist prompt injection》；模拟投资账户是用于解释权限边界的编辑案例。
- 中文按自然口语独立写作，`zh-CN-YunxiaNeural +20%`。

## 视觉与动效

- `1920x1080`、`30fps`、纸灰白板风。
- 文字本体从场景开始即静止可读；唯一文字动效是 `#F4C542` 黄色标注线从零宽度向右划出。
- 图片只为呈现新信息执行一次入场，限左、右、上、下飞入或原地渐显；入场后保持稳定。
- 一个场景里的一个可见元素只执行一次动效。文字、标注线、卡片、边框和图片均不得越出容器或安全区。

## 完成标准

- 成片约两分钟，允许根据最终 TTS 落在 `110-130 秒`。
- 开头问题五秒内念完，首帧完整可见，重点标注线与最终 VTT 对齐，问题念完后保持 `1.2-1.6 秒`。
- 前十二秒出现权威来源与明确结论，前二十秒出现具体损失与收益承诺，前三十秒出现三级权限矩阵并开始解释。
- 脚本逻辑连续、无明显重复、母案例回访间隔不超过三十秒。
- HyperFrames check、DOM 越界、动效白名单、音视频规格、黑帧和静音尾巴检查全部通过。
