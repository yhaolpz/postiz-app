# Tiny Agent landscape long-form character system

## Format

- Canvas: 1920 x 1080, 16:9, 30 fps.
- Target: YouTube long-form explainers viewed on desktop, television, and mobile landscape.
- Sample duration: 27.168 seconds, using a Chinese narration of `增强型大模型：智能体的起点` as a layout and motion proof.

## Visual hierarchy

1. The engineer and Tiny Agent are the main visual subject, not a sidebar beside teaching slides.
2. One continuous task stage supports the full narration: task handoff, retrieval, tool execution, memory, and result.
3. Props enter once and remain in place; narration beats only change character poses and visual focus.
4. On-screen text is limited to the brand, the topic title, and subtitles.
5. A persistent edge-to-edge bottom chapter rail shows only substantive content sections and uses proportional segment widths.

## Style

- Whiteboard background, near-black marker lines, blue structure accents, yellow highlights, and red result emphasis.
- PingFang SC for Chinese headings, labels, and subtitles; Inter remains only for the Tiny Agent brand.
- No source label, chapter label, explanatory note, title underline, subtitle label, progress bar, or capability table.
- Character pose swaps happen only at narration boundaries and use short crossfades without camera movement.
- Retrieval, tool, and memory props stay visible in one spatial scene; inactive groups are dimmed instead of replaced.
- The chapter rail never exposes production beats such as hook, overview, summary, or follow CTA.

## Long-form extension

The same stage can support longer episodes by reusing the character pose library and persistent prop groups. New chapters should introduce a new task situation, not reset into a slide layout.
