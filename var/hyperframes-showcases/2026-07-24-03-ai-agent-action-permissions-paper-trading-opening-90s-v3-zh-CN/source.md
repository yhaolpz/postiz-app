## Confirmed source claims used in this preview

- Prompt injection increasingly resembles social engineering, so defenses cannot rely only on recognizing suspicious wording.
- A robust design limits what untrusted external content can reach and what consequential tools can do.
- Sensitive data transmission and dangerous actions should not happen silently.
- Safeguards can disclose the proposed action and request confirmation, or block it and seek another path.
- Deterministic limits reduce the downside even when a person or an Agent is misled.

## Editorial synthesis

The three-tier Automatic / Confirm / Block matrix is a Tiny Agent teaching tool derived from the source-and-sink model, confirmation and block behavior, and deterministic capability limits. The simulated-investment-account story is an explanatory example, not an OpenAI case and not investment advice.
