## Video direction

Reconstruct one persistent Agent Task Builder interface in the current Tiny Agent whiteboard palette. A visible cursor edits fields, validation changes state, and a compile action produces the result. The interface is the mechanism rather than decoration: no field appears complete before its narration cue, and the result is earned by a visible state transition. Characters remain supporting brand anchors outside the primary UI. No dense code, browser chrome, or tiny text.
