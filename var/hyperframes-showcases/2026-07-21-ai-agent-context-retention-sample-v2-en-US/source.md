## Claims used in this sample

- Context is a finite attention budget with diminishing marginal returns as the context window grows.
- More context can dilute the signal that matters for the next decision.
- The useful target is the smallest sufficient set of high-signal context, not an arbitrarily short prompt.
- The reusable artifact is a six-part pack: outcome, operating rules, tool map, canonical examples, retrieval index, and working state.

## Boundary

The sample does not claim that shorter prompts are always better. It contrasts indiscriminate preloading with sufficient high-signal context and just-in-time retrieval.
