# AI 智能体上下文工程：完整中文成品分镜

## 制作目标

用已确认的留存样片规则制作完整长视频。开头复用正式封面，第一句问完后硬切；正文不使用信息墙，每幕只保留一个主动 Tiny Agent、一个核心物体和一个主判断。

## 开头 48 秒

1. 正式封面从首帧出现，持续到“你知道怎么给 AI 智能体高信号上下文吗？”真实可听结束后的第一个 30fps 帧。
2. Anthropic 方法承诺卡同步显示“成为更擅长使用 AI 的人”和“关注 + 收藏”。
3. 权威结论卡把上下文定义为有限注意力预算。
4. 六部分高信号包提前亮相。
5. 放大的注意力预算装置说明“看到更多，不等于信号更强”。
6. 四字段决策卡收束为目标、约束、证据和当前状态。

## 正文章节

- 注意力预算：完整上下文、注意力稀释、最小充分集合与持续刷新。
- 高信号输入：合适的指令层级、清楚工具契约和少量典型示例。
- 按需检索：轻量引用、渐进式披露、混合策略与检索成本。
- 长任务记忆：压缩、结构化笔记和专门 Agent 的适用边界。
- 上下文包：结果目标、运行规则、工具地图、典型示例、检索索引和工作状态。
- 总结：回收三条核心判断并硬切到固定中文关注结束页。

每个章节的最后三条使用放大的编号标题逐项小结。底部章节导航从正文开始显示，按真实旁白时长分配宽度。

## 固定画面规格

- 1920×1080、30fps、`#ECECEA` 纸灰底、`#111413` 主墨色。
- 主标题默认 92px；长标题 78px；字幕 46px、6px 黑色圆角描边。
- Agent 元素高度 620px；核心物体 480px。
- 只使用硬切、短距离推动和核心物体高亮；长场景每约 5 秒产生一次强调变化。
- 固定中文 CTA 开始时隐藏正文、字幕和章节条，只显示独立结束页。
