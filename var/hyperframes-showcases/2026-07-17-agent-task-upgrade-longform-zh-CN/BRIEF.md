# Tiny Agent 横屏长视频中文版

使用已确认的 Tiny Agent v7 白板表达和固定资产包，生成 1920x1080、30fps 中文横屏讲解视频。工程师和 Tiny Agent 持续参与叙事；中文旁白、字幕、时间轴、场景边界和动画锚点独立生成，固定语速 +50%。

中文版仅保留本地成片。浏览器只允许把视频、文案和封面准备到哔哩哔哩与抖音最终发布按钮之前，禁止由自动化点击发布。

## Intent

帮助观众判断哪些任务值得从聊天升级为 Agent 委托，并交付一张可复用的任务升级判断表。
