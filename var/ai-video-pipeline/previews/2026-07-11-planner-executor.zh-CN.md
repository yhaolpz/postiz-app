# Tiny Agent 中文预览内容

### 2026-07-11 Sat - 规划者和执行者

- Tiny point: 规划和执行是两件不同的事。
- Hook: 别让同一个小机器人同时干所有事。
- Narration: 很多 Agent 系统会把任务拆成两种工作：规划者负责想清楚步骤，执行者一次只完成一步，再把结果回报给规划者。这样听起来慢一点，却能避免一个常见问题：Agent 还没理解任务，就已经开始乱点。规划者先判断顺序，执行者专注完成当前动作。每一步之后都重新沟通，整个系统就更容易检查，也更容易定位错误。
- On-screen text: 先规划，再执行。
- Subtitle blocks:
  - 规划和执行 / 是两件不同的事
  - 规划者决定步骤 / 执行者做完再回报
  - 故意慢一点 / 反而更容易排错
- Keyframes:
  - Tiny Agent wearing a planner hat draws three boxes.
  - Tiny Agent wearing a worker hat checks one box and reports back.
  - Engineer blocks a red button labeled CLICK FIRST.
- TikTok caption: 好的 Agent 会把规划和执行分开。
