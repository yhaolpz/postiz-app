- Narration: What does an agent actually need? Not five frameworks. OpenAI describes three basic parts: a model, tools, and instructions. The model reads the current state, makes decisions, and judges when the work is complete. Tools let the agent gather outside information or change a real system. Instructions define the goal, the expected steps, the boundaries, and what to do when risk appears. Without tools, a model can mostly offer advice. Add tools, and it can update a ticket or calendar. But tools without clear instructions are like giving an intern admin access without explaining the job. Having all three is only the foundation, not a guarantee of reliability. Tiny Rule: the model decides, tools act, and instructions constrain the work. Follow Tiny Agent. Tiny Agent helps you get better at using AI.
