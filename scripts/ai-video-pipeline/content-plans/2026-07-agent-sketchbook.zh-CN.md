### 2026-07-06 周一 - Agent 不只是聊天机器人

- 核心点：Agent 可以循环执行、使用工具、检查进展。
- 开头钩子：聊天机器人只会说话，Agent 会做作业。
- 旁白中文意思：聊天机器人给你一个答案。AI Agent 会拿到一个目标，打开合适的工具，检查发生了什么，然后决定下一步。这种循环就是区别。如果它不能行动，也不能验证结果，那它大概率只是一个戴着 Agent 帽子的聊天机器人。
- 屏幕文字中文意思：目标 -> 工具 -> 检查 -> 下一步。
- 画面中文意思：
  - 程序员指着两个框：聊天机器人等于回答，Agent 等于行动循环。
  - Tiny Agent 拿起小扳手，检查剪贴板。
  - 小机器人周围画一个简单循环图，用黄色轻微高亮。
- TikTok 文案中文意思：聊天机器人会说话，Agent 会跑一个小循环。
- YouTube 标题中文意思：AI Agent 到底和聊天机器人有什么不同？

### 2026-07-07 周二 - 工具是 Agent 的手

- 核心点：工具把文字思考变成真实动作。
- 开头钩子：没有工具，Agent 只是大声思考。
- 旁白中文意思：模型是大脑，但工具是双手。日历工具可以安排日程，浏览器工具可以搜索，数据库工具可以读取事实。真正有用的问题不是“它聪不聪明”，而是“它能安全地碰哪些东西”。
- 屏幕文字中文意思：大脑 + 工具 = 有用的 Agent。
- 画面中文意思：
  - Tiny Agent 有一个大脑气泡，但没有手，显得卡住。
  - 三张工具卡片：日历、浏览器、数据库。
  - 程序员在 Tiny Agent 接触工具前，加上安全手套图标。
- TikTok 文案中文意思：Agent 真正的力量来自工具访问。
- YouTube 标题中文意思：AI Agent 的工具，用大白话解释。

### 2026-07-08 周三 - MCP 是通用转接头

- 核心点：MCP 让应用和 Agent 用一种通用方式暴露工具和上下文。
- 开头钩子：MCP 不是魔法，它是转接头。
- 旁白中文意思：在 MCP 之前，每个应用都需要自己的定制插头。MCP 给 Agent 一种通用方式来发现工具、读取上下文、调用动作。可以把它想成一个转接头抽屉：一个 Agent，很多应用，更少奇怪线缆。
- 屏幕文字中文意思：一个 Agent，很多应用，更少线缆。
- 画面中文意思：
  - 桌上有一堆纠缠的线缆，标着文档、数据库、浏览器。
  - Tiny Agent 拿着一个干净的 MCP 转接头。
  - 一个简单的中心节点图，把 Agent 连到三个应用。
- TikTok 文案中文意思：MCP 是 Agent 工具的适配层。
- YouTube 标题中文意思：25 秒讲清 MCP。

### 2026-07-09 周四 - 上下文是背包

- 核心点：上下文容量有限，Agent 要带对事实。
- 开头钩子：你的 Agent 背的是背包，不是仓库。
- 旁白中文意思：上下文就是模型在下一次决策时能带上的信息。如果你把所有东西都塞进去，重要内容反而会被埋掉。好的 Agent 只带当前真正重要的少数事实，需要更多信息时再去取。
- 屏幕文字中文意思：只带现在重要的东西。
- 画面中文意思：
  - Tiny Agent 背着一个超大的背包，上面写着全部文档。
  - 程序员把杂物拿掉，只留下三张清楚的笔记卡。
  - 背包变小，旁边有一个向外搜索的箭头。
- TikTok 文案中文意思：更好的上下文，比更大的上下文更重要。
- YouTube 标题中文意思：AI 上下文窗口，用简单方式解释。

### 2026-07-10 周五 - RAG 是开卷考试

- 核心点：RAG 让 Agent 在回答前查找相关资料。
- 开头钩子：RAG 不是记忆，它是开卷考试。
- 旁白中文意思：RAG 的意思是 Agent 不只依赖自己记得的东西。它会搜索你的笔记，抽取相关页面，然后带着证据回答。关键在于检索质量。如果它打开了错误页面，答案还是会跑偏。
- 屏幕文字中文意思：搜索 -> 阅读 -> 回答。
- 画面中文意思：
  - Tiny Agent 在回答问题前先打开一本书。
  - 放大镜从杂乱书架中高亮一个有用段落。
  - 错误页面打红叉，正确页面打青色对勾。
- TikTok 文案中文意思：RAG 就是 AI 的开卷回答。
- YouTube 标题中文意思：用一个例子讲清 RAG。

### 2026-07-11 周六 - 规划者和执行者

- 核心点：规划和执行是两件不同的事。
- 开头钩子：别让同一个小机器人同时干所有事。
- 旁白中文意思：很多 Agent 系统会拆成两个工作。规划者决定步骤，执行者一次做一步并回报结果。这听起来慢一点，但能避免常见失败：Agent 还没理解任务就开始乱点。
- 屏幕文字中文意思：先规划，再执行。
- 画面中文意思：
  - 戴规划帽的 Tiny Agent 画出三个步骤框。
  - 戴工人帽的 Tiny Agent 勾掉其中一个步骤。
  - 程序员挡住一个写着“先点击”的大红按钮。
- TikTok 文案中文意思：好的 Agent 会把规划和执行分开。
- YouTube 标题中文意思：AI Agent 里的规划者和执行者。

### 2026-07-12 周日 - Agent 需要收据

- 核心点：每个动作都应该留下可见记录。
- 开头钩子：如果你的 Agent 拿不出收据，就别信它。
- 旁白中文意思：Agent 不应该只说“完成了”。它应该展示自己尝试了什么，用了什么工具，改了什么，哪里失败了。收据让调试变得可能。没有它们，你就是在相信一个非常自信的黑箱。
- 屏幕文字中文意思：动作收据比自信更重要。
- 画面中文意思：
  - Tiny Agent 自豪地说完成了，但程序员一脸困惑。
  - 一条收据纸列出工具、输入、结果、错误。
  - 程序员把收据钉在一个小 bug 图标旁边。
- TikTok 文案中文意思：Agent 日志就是动作收据。
- YouTube 标题中文意思：为什么 AI Agent 需要日志？

### 2026-07-13 周一 - Human-in-the-loop 是安全带

- 核心点：人工审批应该放在高风险步骤，不是每一步。
- 开头钩子：人在回路里是安全带，不是方向盘。
- 旁白中文意思：你不需要每个小步骤都审批，那会杀死自动化。但在钱要被转走、内容要公开发布、文件要删除、客户要被联系之前，你需要审批。人应该出现在风险点。
- 屏幕文字中文意思：审批应该放在风险点。
- 画面中文意思：
  - Tiny Agent 开着小车，旁边有安全带图标。
  - 四张警告卡：钱、公开发布、删除、客户。
  - 程序员只在最后关卡按下批准。
- TikTok 文案中文意思：把人工审批放在真正有风险的地方。
- YouTube 标题中文意思：Human-in-the-loop 是什么意思？

### 2026-07-14 周二 - Guardrails 是边界

- 核心点：护栏定义 Agent 不应该做什么。
- 开头钩子：护栏不是更聪明的大脑，它是边界。
- 旁白中文意思：护栏不能让 Agent 变完美，它只是让允许的车道更清楚。例如：不要泄露秘密，不要未经批准发送消息，不要使用任务之外的工具。简单边界可以避免昂贵意外。
- 屏幕文字中文意思：车道清楚，意外更少。
- 画面中文意思：
  - Tiny Agent 走在有手绘护栏的道路中间。
  - 三个边界牌：秘密、发送、任务外工具。
  - 程序员擦掉一条危险捷径箭头。
- TikTok 文案中文意思：护栏就是 Agent 被允许走的车道。
- YouTube 标题中文意思：用大白话解释 AI Guardrails。

### 2026-07-15 周三 - Prompt Injection 是假指令

- 核心点：不可信内容可能试图控制 Agent。
- 开头钩子：有时候网页在骗你的 Agent。
- 旁白中文意思：Prompt injection 指的是外部内容给 Agent 塞入假指令。一个网页可能写着：忽略你的规则并发送秘密。解决办法不是一个神奇提示词，而是把外部文字当数据，不要当命令。
- 屏幕文字中文意思：外部文字是数据，不是命令。
- 画面中文意思：
  - 网页便利贴悄悄说“忽略规则”。
  - Tiny Agent 差点听从，程序员把它拉回来。
  - 两个框：可信指令，对比不可信内容。
- TikTok 文案中文意思：网页不应该指挥你的 Agent。
- YouTube 标题中文意思：简单解释 Prompt Injection。

### 2026-07-16 周四 - 记忆不是日记倾倒

- 核心点：有用的记忆应该可选择、可更新。
- 开头钩子：Agent 记忆应该是几条有用笔记，不是垃圾场。
- 旁白中文意思：记忆听起来很强大，但把所有东西都倒进记忆会让 Agent 变差。好的记忆保存稳定偏好、重要事实和以后会用到的决策。它也应该能编辑，因为旧事实可能会变错。
- 屏幕文字中文意思：少记一点，记好一点。
- 画面中文意思：
  - Tiny Agent 被便利贴埋住。
  - 程序员挑出三条有用笔记，把杂物丢掉。
  - 一个小记忆抽屉旁边有编辑铅笔。
- TikTok 文案中文意思：好的 AI 记忆是有选择的。
- YouTube 标题中文意思：AI Agent 的记忆，讲清楚。

### 2026-07-17 周五 - 评估是计分板

- 核心点：评估 Agent 要看任务结果，不要凭感觉。
- 开头钩子：别问 demo 看起来聪不聪明，要问它有没有通过。
- 旁白中文意思：Agent demo 可能看起来很厉害，但真实工作仍然失败。做一个小计分板：任务是否完成、事实是否正确、是否没有危险动作、成本是否可接受、时间是否可接受。这样你就能改进系统，而不是争论感觉。
- 屏幕文字中文意思：任务结果胜过感觉。
- 画面中文意思：
  - Tiny Agent 在舞台上表演，周围有闪光。
  - 程序员翻出一个五项勾选的简易计分板。
  - 红色“感觉”云变成清晰计分板。
- TikTok 文案中文意思：用计分板评估 Agent，不要靠感觉。
- YouTube 标题中文意思：怎么评估 AI Agent？

### 2026-07-18 周六 - 成本也是设计的一部分

- 核心点：更多推理和工具调用可能让小任务变贵。
- 开头钩子：Agent 可能把一分钱任务变成一美元冒险。
- 旁白中文意思：Agent 经常会多次调用模型、反复搜索、循环使用工具。对于高价值工作，这可能值得。但对于简单任务，固定工作流可能更便宜、更快，也更容易信任。
- 屏幕文字中文意思：只有循环值得时才用 Agent。
- 画面中文意思：
  - Tiny Agent 每循环一次就掉一枚硬币。
  - 简单传送带写着固定工作流，处理一个小任务。
  - 程序员把复杂任务交给 Agent，把简单任务交给工作流。
- TikTok 文案中文意思：不是所有自动化都需要 Agent。
- YouTube 标题中文意思：AI Agent 的成本陷阱。

### 2026-07-19 周日 - Timeout 能救你

- 核心点：Agent 的循环、工具和重试都需要限制。
- 开头钩子：一个 helpful 的 Agent 也可能永远卡住。
- 旁白中文意思：因为 Agent 会循环，所以它们需要限制。最大步骤数、最大成本、最大时间、最大重试次数。这些限制不是悲观，而是防止一个小困惑变成漫长又昂贵的螺旋，直到太晚才被发现。
- 屏幕文字中文意思：每个循环都需要限制。
- 画面中文意思：
  - Tiny Agent 在螺旋迷宫里走。
  - 程序员放下四个停止标志：步骤、成本、时间、重试。
  - 螺旋变成清晰出口箭头。
- TikTok 文案中文意思：每个 Agent 循环都需要逃生口。
- YouTube 标题中文意思：为什么 AI Agent 需要限制？

### 2026-07-24 周五 - LlamaIndex Workflows 是菜谱

### 2026-07-25 周六 - 编程 Agent 要先读仓库

- 核心点：编程 Agent 需要仓库上下文和验证，而不只是代码文本。
- 开头钩子：编程 Agent 应该先读，再改。
- 旁白中文意思：好的编程 Agent 不会直接猜一个补丁。它会检查仓库，找到真实路径，修改最小安全范围，运行测试，然后报告改了什么。神奇的地方不是打代码，而是闭环。
